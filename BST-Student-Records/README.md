# Binary Search Tree for Student Records

This project implements a Student Record Management System using a Binary Search Tree (BST).

## Student Fields
- Roll Number
- Name
- CGPA

## Features
- Insert Student Record
- Search Student
- Delete Student
- Display Records (Sorted Order)

## Data Structure
Binary Search Tree (BST)

## Compile

gcc student_bst.c -o bst

## Run

./bst
